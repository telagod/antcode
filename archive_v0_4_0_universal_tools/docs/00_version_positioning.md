# 00. v0.3.2 版本定位

## 一句话

> AntCode v0.3.2 = 第一代策略基因变异实验器。

v0.3.1 的关键词是：

```text
信息素改变采样概率
```

v0.3.2 的关键词是：

```text
失败改变策略结构
```

## 为什么不是继续加 UI / 平台 / 插件？

因为 AntCode 的核心风险不是功能少，而是核心反馈闭环还没证明就开始平台化。

v0.3.2 继续遵守反工程化原则：

```text
先证明生命机制，再工程化。
```

## 最小假设

```text
当某个 StrategyGenome 在某类 ExperienceKey 下反复失败时，
系统能够识别 FailureMode，
生成有因果解释的子 StrategyGenome，
并在父子竞赛中验证子策略是否真的更好。
```

## v0.3.2 不声称

- 不声称已经实现真正自主软件工程师。
- 不声称已经拥有通用自我进化能力。
- 不声称可以脱离人类目标和边界。
- 不声称能自动重写项目架构。

它只声称：

```text
策略结构可以被失败反馈修改。
```
